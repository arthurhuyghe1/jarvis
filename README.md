## Assistant IA (LLM)

- Le **LLM tourne directement sur mon PC**
- Il est entraîné / configuré pour :
  - Comprendre mes demandes en langage naturel
  - Les transformer en **instructions structurées**
  - Générer les paramètres nécessaires à la construction des trames réseau

---

## Contrôle réseau & Scapy

La communication avec les périphériques se fait via :
- **Python**
- Le module **Scapy**

Fonctionnalités :
- Construction manuelle de trames réseau
- Envoi de commandes personnalisées
- Compréhension fine des protocoles utilisés par les équipements

Exemple d’usage :
- Piloter une **caméra IP** via des trames construites à la main
- Envoyer des ordres précis sans API propriétaire

---

## Domotique locale

- Ajout prévu de **lumières connectées en WiFi**
- Réseau local isolé via un **routeur Tenda**
- Aucun accès à Internet pour les objets connectés

Avantages :
- Sécurité accrue
- Latence réduite
- Indépendance vis-à-vis des services tiers

---

## Confidentialité & Sécurité

✔ Données traitées uniquement en local  
✔ Pas de cloud  
✔ Pas de collecte externe  
✔ Réseau isolé  

Ce projet répond à une problématique clé :  
> *Comment créer un assistant intelligent sans sacrifier la vie privée ?*
